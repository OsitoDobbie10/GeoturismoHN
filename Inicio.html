Mapas Turisticos,

Mapa de las zonas turisticas de Honduras, localizacion e informacion de los elementos que influyen cada zona de interes.

Geoportal webmapping.

Geportal web que nos permite visualizar y almacenar los datos espaciales del turismo de Honduras.

Base de datos Turismo Honduras.

Generacion de datos fundamentales de turismo, informacion socioeconomica y rutas turisticas almacenados en una base de datos postgis en nuestro geoportal Turismo.

Informacion de nuestras ubicaciones turisticas.

Informacion de interes sobre las zonas turisticas, Hoteles, restaurantes y centros de interes para informacion al publico.

SIG para el turismo
